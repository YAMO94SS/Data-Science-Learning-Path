# Data-Science-Learning-Path
Portafolio de proyectos y ejercicios prácticos de Machine Learning, SQL y Python desarrollados en el Bootcamp de EBAC.
# 📚 Trayectoria en Ciencia de Datos - EBAC

Este repositorio contiene la recopilación de ejercicios, laboratorios y proyectos desarrollados durante mi formación como Científico de Datos. Cubre desde los fundamentos de programación hasta modelos avanzados de Machine Learning.

## 🛠️ Habilidades Desarrolladas
* **Programación:** Python (Ciclos, Funciones, POO, Pandas, Numpy).
* **Machine Learning:** Regresiones, Árboles de Decisión, Clustering (K-Means), PCA y Ensambles.
* **Estadística:** Análisis descriptivo, inferencial y modelos de series de tiempo (ARIMA).
* **Bases de Datos:** SQL avanzado (Joins, Agrupaciones, Particionamiento) e integración con Python.
* **Visualización:** Matplotlib, Seaborn y Power BI.

## 📁 Contenido del Repositorio
* **Machine Learning:** Modelos de clasificación y regresión aplicados.
* **SQL:** Consultas de ingeniería de datos y optimización.
* **Series de Tiempo:** Análisis de tendencia y estacionalidad.
* **Python:** Scripts de automatización y limpieza de datos.
